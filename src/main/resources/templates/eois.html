Expressions of Interest

Glossary of Operations
  • Start chat: Deletes the EOI and redirects you to a chat window with the sender so you can connect.
  • Hide/Unhide: Moves the EOI in/out of the hidden EOIs section below the main EOIs table. If you hide an EOI, you won't see it in the main table and the sender will not be able to send another one until you delete or respond to the hidden EOI. You can un-hide EOIs at any time.
  • Delete: This permanently deletes the EOI. The sender can send you a new EOI any time after deletion
